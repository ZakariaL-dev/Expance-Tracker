## 🌟 Overview
**Expense Tracker** is a powerful personal finance application designed to help users monitor their income and expenses effortlessly. By providing a clear breakdown of balance and transaction history, this tool empowers you to make informed financial decisions through a clean, intuitive interface.

## ✨ Key Features
* **💵 Balance Tracking:** View your total balance, total income, and total expenses at a glance.
* **📝 Transaction History:** Add, view, and delete transactions with clear labeling for positive and negative cash flow.
* **💾 Local Storage:** Your data is automatically saved to your browser, so your records remain intact even after a refresh.
* **📱 Responsive Design:** Manage your budget on the go—optimized for mobile, tablet, and desktop.
* **⚡ Instant Updates:** Real-time calculation of totals as soon as a new entry is added or removed.

## 🚀 Installation
Follow these steps to get a local copy up and running:

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/ZakariaL-dev/Expance-Tracker.git](https://github.com/ZakariaL-dev/Expance-Tracker.git)
    ```
2.  **Navigate to the Project Directory**
    ```bash
    cd Expance-Tracker
    ```
3.  **Launch the App**
    Simply open the `index.html` file in your preferred browser:
    ```bash
    open index.html
    ```

## 📖 Usage
1.  **Add Income:** Enter the text (e.g., "Salary") and a positive amount (e.g., "2000").
2.  **Add Expense:** Enter the text (e.g., "Coffee") and a negative amount (e.g., "-5").
3.  **Review:** Watch the "Your Balance" and "Income/Expense" cards update instantly.
4.  **Delete:** Click the "x" button next to any transaction to remove it from your history and recalculate your balance.
